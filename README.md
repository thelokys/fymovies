## :bulb: Instalação

Abaixe o projeto com

```shell
git clone https://github.com/thelokys/myanimes-app.git # HTTPS
ou
git clone git@github.com:thelokys/myanimes-app.git # SSH
```

Entre da pasta do projeto`cd myanimes-app/`
